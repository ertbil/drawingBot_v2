Zip olarak atılan dosya sağ tıkla ayrılarak ayıklama yapılır
PyCharm açılmalı pycharm açıldıktan sonra hali hazırda açık bir proje
var olması durumuda File> Open Project > "projemizin konumu" açılır

Proje run edilir, gui'de karşımıza çıkan yükle butonuna tıklayarak
çizilmesini istediğimiz verinin yazılı olduğu txt dosyası yüklenir
ve kodun execute edilmesi için 'çiz' butonuna tıklanır.
Tanımlanmış olan olası hatalar, gui'den veyahut terminal'den görülebilir.
